## 4. 패턴 분석 — 개선이 필요한 영역

### 4-A. 과잉 편집 (Precision 문제)

자동이 532초를 잘랐지만 사람과 겹치는 건 115초. **417초가 과잉.**

- **filler 과잉 (113.7s, 적중률 15.7%)**: "네", "어", "맞아요" 같은 짧은 호응을 거의 다 제거. 사람은 자연스러운 대화 흐름 유지를 위해 대부분 남김. → filler 판단 임계값이 너무 낮음
- **manual 과잉 (202.2s, 적중률 15.8%)**: 맥락상 유지해도 되는 부연설명을 "늘어진다"고 판단. 사람은 흐름이 자연스러우면 유지 → manual score 기준이 너무 공격적
- **repetitive 과잉 (75.5s, 적중률 15.3%)**: 대화에서의 약간의 반복은 자연스럽지만 전부 제거 시도

### 4-B. 미탐 (Recall 문제)

사람이 잘른 229초 중 114초를 못 잡음.

- **SRT segment 내부 부분 컷 불가**: 사람 컷 #4,5,9,14,16,17 등은 하나의 SRT segment 안에서 "막 막", "오토노머스, 오토노머스하게" 같은 단어 레벨 반복을 잘라야 함. 현재 podcast-cut은 segment 단위로만 판단
- **SRT에 텍스트가 없는 구간**: #51,53,55 — 의미없는 소리/시행착오가 전사되지 않은 구간. SRT 기반 분석으로는 잡을 수 없음
- **맥락/사실 판단**: #3(블로그 안내=중요도 낮음), #24(정보 부정확), #44(할 이야기 없다) — 텍스트만으로는 유효해 보이지만 사람은 외부 지식으로 판단
- **대화 흐름 판단**: #32(JB의 흐름 끊는 질문), #33(누락 정보 확인 질문) — 대화 맥락에서 빠져도 되는 끼어들기
